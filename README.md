# AI Resume & Portfolio Builder

An AI-powered application that automatically generates tailored resumes, cover letters, and portfolio content based on student data.

## Features

- **AI-Generated Resume Sections**: Professional summaries and accomplishment-driven bullet points
- **Smart Cover Letters**: General and job-specific cover letters
- **Portfolio Content**: Detailed project descriptions for online portfolios
- **Complete Documents**: Full professional document generation

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:

```bash
streamlit run app.py
```

## How It Works

1. Enter your personal information in the sidebar
2. Add your education, skills, experience, and projects
3. Optionally provide a job description for tailored content
4. Click generate buttons to create professional content
5. View and copy the generated content from different tabs

## Project Structure

- `app.py` - Main Streamlit application
- `resume&portfolio.py` - Core logic and functions
- `requirements.txt` - Python dependencies
